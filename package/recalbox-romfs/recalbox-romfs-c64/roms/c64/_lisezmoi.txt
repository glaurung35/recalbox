## RECALBOX - SYSTEM COMMODORE 64 ##

Placez ici vos roms Commodore 64.

Les roms doivent avoir l'extension ".d64/.d71/.d80/.d81/.d82/.g41/.g64/.t64/.x64/.tap/.prg/.p00/.crt/.zip/.7z"

Ce système supporte également les roms compressées au format .zip/.7z.
Attention toutefois, il ne s'agit que d'une archive. Les fichiers contenus dans les .zip/.7z doivent correspondre aux extensions citées précédemment.
De plus, chaque fichier .zip/.7z ne doit contenir qu'une seule rom compressée.


Mapping manette :

Start = gui menu
Select = switch souris/joystick
Y = clavier virtuel
L2 = switch Joy1/Joy2
