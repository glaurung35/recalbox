## RECALBOX - SYSTEM FBNEO ##

Placez ici vos roms FinalBurn Neo (support cps3, version plus recente, mais moins performant que piFBA).

La version des roms doit etre 1.0.0.01

Les roms doivent avoir l'extension ".iso/.zip/.7z"

Fichiers spéciaux pour le core fbneo :
- Ajoutez vos fichiers samples dans le répertoire /recalbox/share/bios/fbneo/samples/

Consultez la documentation à cette adresse :
https://recalbox.gitbook.io/documentation/v/francais/emulateurs/arcade/fbneo
